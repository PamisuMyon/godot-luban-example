# godot-luban-example
Example of using Luban with Godot 4.x. 在Godot 4.x中使用Luban的简单示例.

Branches:
- classic: Luban classic
- next: Luban Next

classic分支的scriban模板文件(tpl)有部分修改。Next分支暂不支持多维数组。

```
.
|-- godot-luban-project   Godot项目目录
|   |-- ...
`-- godot-luban-data 配置表目录
    |-- Configs 示例配置表及导表脚本
    `-- Luban   导表工具

```
